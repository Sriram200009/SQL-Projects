# SQL Murder Mystery – Solution

This repository contains my solution to the **SQL Murder Mystery** challenge, an interactive puzzle where you solve a crime using SQL queries.

## 🕵️ Problem Statement
A murder occurred in SQL City on **January 15, 2018**. Using SQL queries, I investigated witnesses, interviews, and records to uncover the killer and the mastermind.

## 🔎 Approach
1. Identified witnesses from the crime scene report.
2. Retrieved their addresses and interview transcripts.
3. Narrowed suspects using gym membership and car license data.
4. Found the hired killer (Jeremy Bowers).
5. Discovered the true mastermind (Miranda Priestly).

## 📂 Repository Structure
- `queries/` → SQL scripts for each step.
- `results/` → Output tables and findings.
- `README.md` → Documentation of the process.

## ✅ Outcome
- **Murderer:** Jeremy Bowers  
- **Mastermind:** Miranda Priestly  

## 🎯 Learnings
- Practiced advanced SQL joins, filtering, and aggregation.
- Translated narrative clues into database logic.
- Demonstrated structured problem‑solving with SQL.
